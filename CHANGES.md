# 変更履歴

## README統合 (2025-11-02)

### 変更内容

**統合前**:
- `README.md` - メインREADME
- `INSTALL.md` - インストール方法
- `README_SETUP.md` - セットアップ方法

**統合後**:
- `README.md` - 全ての情報を統合した単一のREADME

### 新しいREADME.mdの構成

1. **概要** - プロジェクトの説明
2. **今すぐ試す** - デモの実行方法
3. **機能** - 主要機能の紹介
4. **インストール** - 3つのインストール方法
   - プラグインマネージャー使用
   - 手動インストール
   - Vimディレクトリへコピー
5. **使い方** - 基本操作と例
6. **インストール確認** - 正しく設定されているか確認
7. **テスト** - テストの実行方法
8. **トラブルシューティング** - よくある問題と解決方法
9. **ファイル構成** - プロジェクト構造
10. **ヒント** - 便利な情報
11. **コントリビューション** - 貢献方法
12. **ライセンス** - MIT License

### バックアップ

古いファイルはバックアップとして保存されています：
- `INSTALL.md.bak`
- `README_SETUP.md.bak`

これらは `.gitignore` に追加されており、Gitには含まれません。

## 主な改善点

- **一元化**: 全ての情報が1つのファイルに
- **見やすさ**: 絵文字を使用したセクション区切り
- **完全性**: インストールからトラブルシューティングまで網羅
- **実用性**: コピー＆ペースト可能なコード例
- **明確性**: パスは抽象化し、実例も提供

## ドキュメント構成（最終版）

```
sphinx-vim/
├── README.md              # メインドキュメント（統合版）
├── QUICKSTART.md          # クイックスタート
├── USAGE.md               # 詳細な使用方法
├── TROUBLESHOOTING.md     # トラブルシューティング詳細
└── SUMMARY.md             # 全体の変更まとめ
```

簡潔な情報は README.md に、詳細情報は各専門ドキュメントに分離。
