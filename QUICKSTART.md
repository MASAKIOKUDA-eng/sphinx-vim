# 🚀 クイックスタート

## 最も簡単な方法: デモスクリプトを使う

```bash
vim -u demo.vim
```

これで対話的なデモが起動します。画面の指示に従ってください。

## Vim の基本操作（重要！）

Vimには2つの主要なモードがあります：

### 1. NORMALモード（コマンドモード）
- Vimを起動すると、最初はこのモード
- カーソル移動やコマンド実行ができる
- **画面下部に何も表示されていない状態**
- Escキーを押すといつでもこのモードに戻れる

### 2. INSERTモード（挿入モード）
- 文字を入力できるモード
- **画面下部に `-- INSERT --` または `-- 挿入 --` と表示される**
- NORMALモードで `i` キーを押すと入る

## 📝 ステップバイステップ手順

### 方法1: デモスクリプトで試す（推奨）

```bash
# コマンドプロンプトまたはPowerShellで実行
vim -u demo.vim
```

1. 起動すると説明が表示される → **何か任意のキーを押す**
2. デモファイルが開く
3. カーソルが `.. cod` の行にある
4. **`i` キーを押す** → 画面下部に `-- INSERT --` が表示される
5. **Ctrl キーを押しながら X を押す**
6. **Ctrl キーを押しながら O を押す**
7. 補完メニューが表示される！
   - `code-block::`
   - `codeauthor::`
8. ↓↑キーまたは Ctrl-N/Ctrl-P で選択
9. Enter で確定
10. **Esc キーを押す** → NORMALモードに戻る

### 方法2: example.rst で試す

```bash
vim example.rst
```

1. 最終行に移動:
   - NORMALモードで `G` キーを押す（最終行へ移動）
   - または `:$` と入力してEnter

2. 行末に移動:
   - NORMALモードで `$` キーを押す

3. INSERTモードに入る:
   - **`i` キーを押す**
   - 画面下部に `-- INSERT --` が表示されることを確認

4. 補完を試す:
   - `cod` と入力
   - **Ctrl-X Ctrl-O** を押す
   - 補完メニューが表示される

### 方法3: 新しいファイルで試す

```bash
vim test.rst
```

1. **`i` キーを押す** → INSERTモードに入る
2. 以下を入力:
   ```
   Test
   ====

   .. cod
   ```
3. **Ctrl-X Ctrl-O** を押す
4. 補完メニューが表示される！

## 🎮 キーボード操作まとめ

### モード切り替え
- `i` = INSERTモードに入る（NORMALモードで押す）
- `Esc` = NORMALモードに戻る（いつでも）

### 補完操作（INSERTモードで）
- `Ctrl-X Ctrl-O` = 補完開始
- `Ctrl-N` = 次の候補
- `Ctrl-P` = 前の候補
- `↓` `↑` = 候補移動
- `Enter` = 確定
- `Esc` = キャンセル

### Vimの終了（NORMALモードで）
- `:q` + Enter = 終了（変更がない場合）
- `:q!` + Enter = 強制終了（変更を破棄）
- `:wq` + Enter = 保存して終了

## ❓ トラブルシューティング

### Q: 「挿入モードにならない」
**A:** `i` キーを押していますか？画面下部に `-- INSERT --` が表示されるか確認してください。

### Q: 「何も起こらない」
**A:**
1. まず Esc キーを押してNORMALモードに戻る
2. `:set filetype?` と入力して Enter → `rst` と表示されるか確認
3. もう一度 `i` キーでINSERTモードに入る

### Q: 「Ctrl-X Ctrl-O が動かない」
**A:**
1. INSERTモードになっているか確認（画面下部に `-- INSERT --` があるか）
2. `.. ` で始まる行にカーソルがあるか確認
3. Ctrlキーを**押したまま** X を押し、その後 O を押す

### Q: 「文字が入力できない」
**A:** NORMALモードになっています。`i` キーを押してINSERTモードに入ってください。
